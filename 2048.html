                            About This Game



                                                                    2048 is a single-player sliding the puzzel video game written by Italian web
                                       developer Gabriele Cirulli and published on GitHub. The objective of the
                                       game is to slide numbered tiles on a grid to combine them to creat a tile
                                       with the number 2048; however, one can continue to play the game after
                                       reaching the goal, creating tiles with larger numbers. It was originally
                                       written in Java and CSS over a weekend, and released on 9 March
                                       2014 as free and open-source software subject to the MT Lience.
                                       Version for Ios and Android followed in May 2014.


                                                                    2048 is played on a plain 4X4 grid, with numbered tiles that slide when a
                                       player moves them using the four arrows keys. Every turn, a new tile
                                       randomly appears in an empty spot on the board with a value of either
                                       2 or 4. Tiles slide as far as possible in the chosen direction until they
                                       are stopped by either another tile or the edge of the gride. If two tiles
                                       of the same number collide while moving, they will merge into a tile
                                       with the total value of the two tiles that collided. The resulting tile
                                       cannot merge with another tile again in the same move. Higher-
                                       scoring tiles emit a soft glow, the hieghest possible tile is 131,072.


                                                                    Strategies in 2048 include keeping the hieghest tiles in a specific corner
                                       and to keep that tile in that corner and to fill the specified row with the
                                       highest numbers.
